Atualização do site público com screenshots reais.

Arquivos:
- assets/screenshot-main.png
- assets/screenshot-history.png
- INDEX_SCREENSHOTS_PATCH.html
- CSS_SCREENSHOTS_PATCH.css

Como aplicar:
1. Copie os PNGs para a pasta assets do repositório velocimetro-site.
2. No index.html, substitua a seção de screenshots pelo conteúdo de INDEX_SCREENSHOTS_PATCH.html.
3. No assets/styles.css, cole o conteúdo de CSS_SCREENSHOTS_PATCH.css no final.
4. Faça commit e push.
