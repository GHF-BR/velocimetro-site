# Velocímetro Pro GPS — site público

Este repositório contém a página pública, a política de privacidade e materiais de divulgação do app Velocímetro Pro GPS / GPS Speedometer Pro.

Arquivos principais:

- `index.html`: página inicial do app
- `privacy.html`: política de privacidade bilíngue
- `assets/styles.css`: estilo visual da página
- `.nojekyll`: evita processamento Jekyll no GitHub Pages

## Publicação no GitHub Pages

No GitHub:

1. Acesse o repositório `velocimetro-site`
2. Vá em Settings
3. Clique em Pages
4. Em Build and deployment, selecione Deploy from a branch
5. Branch: `main`
6. Folder: `/root`
7. Save

URL esperada:

- Site: `https://ghf-br.github.io/velocimetro-site/`
- Política: `https://ghf-br.github.io/velocimetro-site/privacy.html`
